# Java 注释
> Javadoc 可以生成 Java API 开发文档。好的文档注释能够帮助别人来迅速理解你写的代码，提升代码的可阅读性，可维护性。
> 
> 添加有代码注释与无代码注释的对比，让大家感受一下
    

    
## 行注释
 ## 1. 单行注释
 ```java
 public void test(){
    // 调用外部接口获取信息
    this.queryxx();
}
 ```
 用于简单的描述一行代码的功能
 
 ### 2. 多行注释
 ```java
 public void test2(){
    /**
     * 这段代码涉及逻辑比较复杂
     * 先查询，在生成数据
     */
    this.queryxx();
}
 ```  
 用于详细的描述一行或一段代码的功能
 
## 文档注释
```java
/**
 * 描述部分
 * 
 * 标记部分
 */
```
